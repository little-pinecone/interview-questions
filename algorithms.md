# Time complexity of array sorting algorithms

| Algorithm      | Worst case scenario                |
|:---------------|:-----------------------------------|
| Quicksort      | *O(n^2)*{: .text-red-200 }         |
| Mergesort      | *O(n log(n))*{: .text-yellow-200 } |
| Timsort        | *O(n log(n))*{: .text-yellow-200 } |
| Heapsort       | *O(n log(n))*{: .text-yellow-200 } |
| Bubble Sort    | *O(n^2)*{: .text-red-200 }         |
| Insertion Sort | *O(n^2)*{: .text-red-200 }         |
| Selection Sort | *O(n^2)*{: .text-red-200 }         |
| Tree Sort      | *O(n^2)*{: .text-red-200 }         |
| Shell Sort     | *O(n(log(n))^2)*{: .text-red-200 } |
| Bucket Sort    | *O(n^2)*{: .text-red-200 }         |
| Radix Sort     | *O(nk)*{: .text-green-000 }        |
| Counting Sort  | *O(n+k)*{: .text-green-000 }       |
| Cubesort       | *O(n log(n))*{: .text-yellow-200 } |
